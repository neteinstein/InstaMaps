# InstaMaps

## Description
InstagramToGoogleMaps is an Android application that integrates with Instagram to extract location data from your posts and visualize them on Google Maps. It helps users track and explore places they've visited based on their Instagram activity, providing a personalized map view of their adventures.

## Features
- Fetch location-tagged posts from Instagram.
- Display locations on an interactive Google Maps interface.
- Filter posts by date or location.
- Export map data for offline use.

## Requirements
- Android device running API level 21 or higher.
- Instagram account with location-enabled posts.
- Google Maps API key (configure in the app settings).

## Installation
1. Clone the repository: `git clone https://github.com/neteinstein/InstagramToGoogleMaps.git`
2. Open the project in Android Studio.
3. Build and run the app on an emulator or device using Gradle.

## Usage
1. Launch the app and log in with your Instagram credentials.
2. Grant necessary permissions for location access.
3. The app will fetch your location-tagged posts and display them on the map.
4. Tap on map markers to view post details.
5. Use the filter options to narrow down results.
